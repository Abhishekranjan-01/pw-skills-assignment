What is HTML?

HTML stands for Hypertext Markup Language. It is the standard markup language used for creating web pages and applications. HTML provides a structured framework for defining the content and layout of a webpage, including text, images, links, tables, forms, and various other elements.


HTML documents consist of a series of elements, which are represented by tags. Tags are enclosed in angle brackets and appear as pairs, consisting of an opening tag and a closing tag. The opening tag denotes the start of an element, and the closing tag denotes the end of that element. The content between the tags represents the information or structure associated with the element.




What is semantic HTML ?

Semantic HTML refers to the use of HTML markup elements that convey meaning and structure to both the browser and the developer. It involves using HTML elements that accurately represent the content and purpose of the information they enclose. By using semantic HTML, you can enhance the accessibility, search engine optimization (SEO), and maintainability of your web pages.


Semantic HTML, combined with proper CSS styling and accessibility considerations, can result in more robust and maintainable web pages that provide a better user experience and are easier to understand and modify for developers.


List of commonly use semantic

  1. header: Represents the introductory content or a container for a set of introductory content at the beginning of a page or section.
  2. nav: Represents a section of a page that contains navigation links.
  3. main: Represents the main content of a document. It should be unique within the document and not nested within other semantic elements.
  4. article: Represents a self-contained composition in a document, such as a blog post or news article.
  5. section: Represents a standalone section of content within a document, such as a chapter or tabbed content.
  6. aside: Represents content that is tangentially related to the main content, such as sidebars or pull quotes.
  7. footer: Represents the footer of a section or the whole document, typically containing information about the author, copyright, or contact details.
